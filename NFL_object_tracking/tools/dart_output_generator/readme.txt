
Use the MS Excel file dart_message_generator.xlsx
to generate DART messages for a simulation of 
Two Players with two tags each, 
runing along a line.

Copy paste the data into a text file.

In a Linux system, use teh Phyton program   
convert_to_dart_format.py
to convert the MS Exel data with '|' message separator,
into a correct DART ouput format with \n line separators.
